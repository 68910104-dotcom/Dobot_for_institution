========================================================================
                     DOBOT COLOR SORTING PROJECT
========================================================================

[ รายละเอียดโปรเจกต์ ]
โปรเจกต์นี้ใช้แขนกล Dobot Magician ร่วมกับ Computer Vision (OpenCV) 
เพื่อคัดแยกวัตถุตามสีอัตโนมัติ โดยระบบจะหยิบวัตถุจากจุดรับ ไปตรวจสอบสี
ผ่านกล้อง และนำไปวางในตำแหน่งที่ถูกต้อง (แดง, เขียว, น้ำเงิน)

------------------------------------------------------------------------
[ ความสามารถของระบบ ]
------------------------------------------------------------------------
1. Auto-Port Detection: 
   - ค้นหาและเชื่อมต่อกับ Dobot ผ่านพอร์ต USB อัตโนมัติ

2. Color Detection: 
   - ตรวจจับสี (Red, Green, Blue) ด้วยระบบ HSV 
   - มีการเปิด-ปิดกล้องเฉพาะตอนใช้งานเพื่อประหยัดทรัพยากรเครื่อง

3. Sorting Logic: 
   - หยิบวัตถุจาก 3 ตำแหน่งที่กำหนด
   - ถ้าเป็นสี แดง / เขียว / น้ำเงิน: แยกไปลงกล่องตามสีนั้นๆ
   - ถ้า ไม่รู้จักสี: นำกลับไปวางที่เดิม

4. Manual Control: 
   - ควบคุมการเริ่มรอบการทำงานผ่านคีย์บอร์ด

------------------------------------------------------------------------
[ ความต้องการระบบ (Requirements) ]
------------------------------------------------------------------------
A. ฮาร์ดแวร์ (Hardware)
   1. แขนกล Dobot Magician + หัวดูด (Suction Cup)
   2. กล้อง Webcam (ติดตั้งในมุมที่แขนกลสามารถยกของมาโชว์ได้)
   3. วัตถุสีต่างๆ (แดง, เขียว, น้ำเงิน)

B. ซอฟต์แวร์ (Software & Libraries)
   1. Python 3.x
   2. Libraries ที่ต้องติดตั้ง:
      - opencv-python
      - numpy
      - pydobot
      - pyserial

   (คำสั่งติดตั้ง: pip install opencv-python numpy pydobot pyserial)

------------------------------------------------------------------------
[ การตั้งค่าก่อนใช้งาน (Configuration) ]
------------------------------------------------------------------------
ก่อนรันโปรแกรม ให้แก้ไขค่าพิกัดในไฟล์ "Color_sorting.py.py" ให้ตรงกับ
สภาพแวดล้อมจริงของคุณ:

1. pickup_points: พิกัด x, y, z ของวัตถุต้นทาง (รองรับ 3 จุด)
2. camera_point:  พิกัดหน้ากล้องสำหรับยกวัตถุไปเช็คสี
3. drop_points:   พิกัดสำหรับวางวัตถุแยกตามสี (Red, Green, Blue)

------------------------------------------------------------------------
[ วิธีการใช้งาน (How to Use) ]
------------------------------------------------------------------------
1. เชื่อมต่อสาย USB ของ Dobot และ Webcam เข้ากับคอมพิวเตอร์
2. รันโปรแกรมด้วย Python:
   python Color_sorting.py.py

3. เมื่อโปรแกรมเชื่อมต่อสำเร็จและเคลื่อนไปที่จุดเริ่มต้น จะมีเมนูขึ้น:
   - กด [1] แล้ว Enter: เพื่อเริ่มกระบวนการคัดแยก (ทำครบทั้ง 3 จุดรับของ)
   - กด [q] แล้ว Enter: เพื่อจบการทำงานและปิดโปรแกรม

------------------------------------------------------------------------
[ การแก้ปัญหาเบื้องต้น (Troubleshooting) ]
------------------------------------------------------------------------
* Error: Dobot not found
  - ตรวจสอบสาย USB 
  - เช็คว่ามีโปรแกรมอื่นแย่งใช้ Port อยู่หรือไม่

* กล้องไม่เปิด / จอดำ
  - ลองเปลี่ยน index ของกล้องในโค้ดจาก cv2.VideoCapture(1) เป็น 0 หรือ 2

* สีผิดเพี้ยน
  - แสงสว่างมีผลต่อการตรวจจับ HSV ให้ลองปรับค่า Hue ในฟังก์ชัน 
    detect_color_from_frame หรือจัดแสงให้สม่ำเสมอ
