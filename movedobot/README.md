# Dobot Web Control Interface

**Dobot Web Control Interface** คือระบบซอฟต์แวร์สำหรับควบคุมและบริหารจัดการการทำงานของแขนกลหุ่นยนต์ Dobot Magician ผ่านเครือข่ายคอมพิวเตอร์ (Web-based Application) พัฒนาด้วยภาษา Python โดยใช้ Flask Framework เป็นแกนหลักในการประมวลผลฝั่งเซิร์ฟเวอร์ และออกแบบส่วนติดต่อผู้ใช้งาน (User Interface) ให้มีความทันสมัยและรองรับการใช้งานในรูปแบบ Dark Mode เพื่อความสะดวกในการปฏิบัติงาน

## คุณสมบัติหลักของระบบ (Key Features)

1.  **Industrial Motion Control:** รองรับการควบคุมการเคลื่อนที่ของแขนกลทั้ง 4 แกน (X, Y, Z และ R) รวมถึงการควบคุมอุปกรณ์ปลายมือ (End Effector) ประเภท Suction Cup
2.  **Real-time Telemetry Monitoring:** แสดงผลข้อมูลพิกัดปัจจุบันของหุ่นยนต์ (X, Y, Z, R) แบบเรียลไทม์ ผ่านการรับส่งข้อมูลอัตโนมัติจากเซิร์ฟเวอร์
3.  **Coordinate Precision Command:** รองรับการสั่งการเคลื่อนที่โดยการระบุพิกัดทางคณิตศาสตร์ที่แม่นยำ (Go-to Coordinate)
4.  **Keyboard Shortcut Integration:** เพิ่มประสิทธิภาพการทำงานด้วยระบบคีย์ลัด สั่งการผ่านแป้นพิมพ์คอมพิวเตอร์
5.  **Responsive Design:** อินเตอร์เฟซถูกออกแบบให้รองรับการแสดงผลบนหน้าจอขนาดต่าง ๆ โดยใช้ Tailwind CSS

## ความต้องการของระบบ (System Requirements)

เพื่อให้ซอฟต์แวร์ทำงานได้อย่างสมบูรณ์ ระบบคอมพิวเตอร์จำเป็นต้องมีองค์ประกอบดังนี้:

* **ฮาร์ดแวร์:**
    * หุ่นยนต์ Dobot Magician
    * สายเชื่อมต่อ USB
* **ซอฟต์แวร์:**
    * Python เวอร์ชัน 3.6 หรือสูงกว่า
    * ไดรเวอร์ Silicon Labs CP210x USB to UART Bridge (สำหรับการเชื่อมต่อ Serial Port)

## การติดตั้ง (Installation)

1.  **ดาวน์โหลดซอร์สโค้ด**
    นำไฟล์โปรเจกต์ทั้งหมด (ประกอบด้วย `app.py` และโฟลเดอร์ `templates` ที่บรรจุไฟล์ `index.html`) ลงในไดเรกทอรีที่ต้องการ

2.  **ติดตั้งไลบรารีที่จำเป็น**
    เปิด Command Prompt หรือ Terminal และดำเนินการคำสั่งต่อไปนี้เพื่อติดตั้ง Python Packages ที่ระบุ:

    ```bash
    pip install flask pydobot pyserial
    ```

## การตั้งค่าการเชื่อมต่อ (Configuration)

ก่อนเริ่มต้นใช้งาน จำเป็นต้องระบุหมายเลขพอร์ตการเชื่อมต่อ (COM Port) ให้ถูกต้องตามที่ระบบปฏิบัติการกำหนดให้กับอุปกรณ์ Dobot

1.  ตรวจสอบหมายเลขพอร์ตที่ **Device Manager** (สำหรับ Windows) ภายใต้หัวข้อ *Ports (COM & LPT)*
2.  เปิดไฟล์ `app.py` ด้วยโปรแกรมแก้ไขข้อความ (Text Editor)
3.  แก้ไขตัวแปร `MANUAL_PORT` ในบรรทัดที่ 9 ให้ตรงกับพอร์ตของท่าน

    ```python
    # ตัวอย่างการตั้งค่าในไฟล์ app.py
    MANUAL_PORT = 'COM3'  # เปลี่ยนเป็น COM3, COM4 หรือ COM5 ตามความเป็นจริง
    ```

## การเริ่มต้นใช้งาน (Usage)

1.  **ประมวลผลโปรแกรม**
    สั่งรันเซิร์ฟเวอร์ด้วยคำสั่ง:
    ```bash
    python app.py
    ```
    *หมายเหตุ: หากการเชื่อมต่อสำเร็จ ระบบจะแสดงข้อความ "CONNECTED SUCCESS" ในหน้าต่าง Terminal*

2.  **เข้าสู่ระบบควบคุม**
    เปิดเว็บเบราว์เซอร์และเข้าถึงที่อยู่:
    * `http://localhost:5001`

## คู่มือการควบคุม (Operation Guide)

### ส่วนควบคุมผ่านหน้าจอ (Graphical Interface)

* **Z-Axis Lift:** ปุ่มสำหรับควบคุมระดับความสูงของหัวจับ (ขึ้น/ลง)
* **XY-Plane Jog:** แผงควบคุมทิศทางสำหรับการเคลื่อนที่ในระนาบแนวนอน
* **End Effector:** ปุ่มสำหรับเปิด/ปิด ระบบปั๊มลม (Suction)
* **Target Coordinate:** ช่องป้อนค่าตัวเลขสำหรับการสั่งเคลื่อนที่ไปยังพิกัดที่กำหนด

### ส่วนควบคุมผ่านแป้นพิมพ์ (Keyboard Shortcuts)

| ปุ่มกด | คำสั่งการทำงาน |
| :--- | :--- |
| **ลูกศรขึ้น (↑)** | เคลื่อนที่ไปด้านหน้า (แกน X+) |
| **ลูกศรลง (↓)** | เคลื่อนที่ไปด้านหลัง (แกน X-) |
| **ลูกศรซ้าย (←)** | เคลื่อนที่ไปทางซ้าย (แกน Y-) |
| **ลูกศรขวา (→)** | เคลื่อนที่ไปทางขวา (แกน Y+) |
| **W** | ยกแกน Z ขึ้น |
| **S** | ลดแกน Z ลง |

## การแก้ไขปัญหาเบื้องต้น (Troubleshooting)

* **ไม่สามารถเชื่อมต่ออุปกรณ์ได้ (Connection Failed):**
    * ตรวจสอบว่าสาย USB เชื่อมต่อแน่นหนาหรือไม่
    * ตรวจสอบว่าโปรแกรม Dobot Studio หรือโปรแกรมอื่นไม่ได้ใช้งานพอร์ตดังกล่าวอยู่
    * ตรวจสอบความถูกต้องของ `MANUAL_PORT` ในไฟล์ `app.py`
* **ข้อผิดพลาดไลบรารี (Import Error):**
    * ตรวจสอบให้แน่ใจว่าได้ติดตั้งไลบรารี `pydobot` และ `pyserial` เรียบร้อยแล้ว
